# Mi Web

Proyecto de portafolio simple con HTML, CSS y JavaScript.

Cómo ver el proyecto:

- Abre `index.html` en un navegador.
- O habilita GitHub Pages en la rama `main` para publicarlo desde GitHub.

Contenido:

- `index.html` — página principal
- `style.css` — estilos
- `script.js` — scripts
