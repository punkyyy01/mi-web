Sobre mí

Soy Frambuesa: diseñador y desarrollador web apasionado por interfaces limpias, accesibles y con buen rendimiento. Me especializo en crear productos que transmiten confianza y resolución de problemas reales para usuarios y equipos.

Proyectos destacados

Purgatory — Página web

Rediseñé y desarrollé la web de Purgatory, enfocándome en experiencia móvil, accesibilidad y SEO técnico. Implementé un diseño modular con componentes reutilizables y optimizaciones de carga (lazy-load, imágenes responsive). Proyecto realizado íntegramente por mí desde cero.

Tecnologías: HTML, CSS moderno, JavaScript ligero, optimización de imágenes.

ERP — Sistema de gestión

Desarrollé íntegramente el panel ERP para gestión de inventario, facturación y usuarios. Implementé las interfaces de datos, formularios robustos y flujos de trabajo desde cero, reduciendo errores operativos.

Tecnologías: Frontend modular, integración con API REST, autenticación y roles.

Tienda de muebles — Plataforma de venta

Desarrollé de forma independiente la tienda online para una marca de muebles: catálogo, carrito, proceso de checkout y optimizaciones de conversión. Proyecto construido por mí desde cero, con especial foco en rendimiento y accesibilidad móvil.

Tecnologías: HTML/CSS, JS, integración con pasarela de pago y servidor para inventario.
